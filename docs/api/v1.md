# API V1

## `POST /api/v1/sign-up`

Creates a new user account with an email and password.

### Request

- Authentication: none
- Content-Type: `application/json`

Schema:

```json
{
  "email": "string",
  "password": "string"
}
```

Example:

```json
{
  "email": "user@example.com",
  "password": "super-secret"
}
```

### Response

Success:

- Status: `201 Created`

Schema:

```json
{
  "id": 1,
  "email": "string",
  "token": "string"
}
```

Conflict:

- Status: `409 Conflict`

Schema:

```json
{
  "detail": "User with this email already exists"
}
```

## `POST /api/v1/sign-in`

Verifies user credentials and returns the matching user record when authentication succeeds.

### Request

- Authentication: none
- Content-Type: `application/json`

Schema:

```json
{
  "email": "string",
  "password": "string"
}
```

Example:

```json
{
  "email": "user@example.com",
  "password": "super-secret"
}
```

### Response

Success:

- Status: `200 OK`

Schema:

```json
{
  "id": 1,
  "email": "string",
  "token": "string"
}
```

Unauthorized:

- Status: `401 Unauthorized`

Schema:

```json
{
  "detail": "Invalid email or password"
}
```
